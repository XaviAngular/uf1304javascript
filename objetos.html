Variables de tipo Objeto JavaScript

Las variables de tipo objeto son variables que permiten almacenar multiples valores en una sola variable. Por ejemplo, si el objetivo es almacenar los datos de un trabajador, seguramente necesitamos trabajar con varios elementos dentro de un solo objeto, un para almacenar el nombre, otro para la edad, otro para el departamento, otro para el email, etc.
Las variables de tipo objeto se definen por pares, uno define la etiqueta/campo y el otro el valor. Por ejemplo:
var trabajador1={nombre:'Manuel',edad:30,departamento:'contabilidad', email:'manuel@empresa.com'};
Para acceder a un elemento del objeto, se declara el objeto y se añade un punto y el nombre del campo/etiqueta al que se desea acceder y retorna el valor. Ejemplo
console.log(trabajador1.email);

Modificar elementos de un objeto

Para modificar un valor de un objeto, basta con igualarlo al nuevo valor. Siguiendo el ejemplo anterior, para modificar la edad del trabajador1 a 40 sería:
trabajador1.edad=40;

Recorrer Objeto con jQuery

Para recorrer y mostrar cada uno de los nombre de campo o etiquetas con sus valores, uno de los método que mejor funcionaria sería el $.each de jQuery. Ejemplo:
$.each(trabajador1, function(k,v){ console.log(k); console.log(v); });
